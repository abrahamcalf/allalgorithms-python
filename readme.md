## Install

```
pip install allalgorithms
```

## Usage Example

```py
from allalgorithms.searches import binary_search

arr = [-2, 1, 2, 7, 10, 77]

print(binary_search(arr, 7))
# -> 3

print(binary_search(arr, 3))
# -> None
```
